# 日本語 語彙 & 接続詞 (JLPT Prep)

このノートは、JLPT（日本語能力試験）対策として、高得点につながる語彙、コロケーション、および文章の論理的流れを作る接続詞を記録するためのものです。

---

## 🔗 接続詞 (論理的な文章構成)

### 1. 情報の追加 / サポート
*   **さらに** (例: *さらに、留学プログラムは自立心を養います。*)
*   **そのうえ**
*   **それに加えて**

### 2. 対比 & 逆接 (対比と譲歩)
*   **一方で** (対比する場合に使用。例: *都市部が発展する一方で、地方の過疎化が進んでいます。*)
*   **しかし / それに対して / 反対に**
*   **にもかかわらず** (例: *高コストであるにもかかわらず、公共交通機関の利用率は低いままです。*)

### 3. 因果関係 (原因と結果)
*   **したがって / その結果 / ゆえに**
*   **〜に起因する** (例: *これらの問題は、コミュニケーション不足に起因することが多い。*)
*   **〜への道を開く / 〜のきっかけとなる** (例: *技術の進歩は、教育改革への道を開きます。*)

### 4. 例示
*   **例えば**
*   **具体的に言うと**
*   **〜が好例である** (例: *自転車利用が普及しているコペンハーゲン市が好例である。*)

## 📚 アカデミック & 言い換え表現
単調な言葉遣いを避け、より表現力豊かな表現を練習しましょう。

| 基本の言葉 | 高度な表現 | 例文 |
| :--- | :--- | :--- |
| **良い** | 有益な、肯定的な、効果的な | *この政策は地域社会にとって極めて有益であることが証明された。* |
| **悪い** | 有害な、不利な、悪影響を及ぼす | *睡眠不足は認知機能に有害な影響を及ぼします。* |
| **多い** | 無数の、膨大な、多数の | *現代の消費者には無数の選択肢が存在します。* |
| **悪くなる** | 悪化する、拗れる | *構造的な問題を放置すれば、事態はさらに悪化するだけだ。* |
| **解決する** | 緩和する、解消する、対処する | *政府は気候変動リスクを緩和するため、迅速に行動しなければならない。* |
| **重要** | 極めて重要な、不可欠な、不可避の | *批判的思考力を養うことは、学生にとって極めて重要です。* |
| **使う** | 利用する、活用する、適用する | *学校は生徒の学習意欲を高めるために、インタラクティブな技術を活用すべきです。* |
